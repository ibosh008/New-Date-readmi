# New-Date-readmi

## GET Methods

GET Methods барои гирифтани маълумот аз Date Object истифода мешаванд.

- getFullYear() → гирифтани сол
- getMonth() → гирифтани моҳ (0-11)
- getDate() → гирифтани рӯзи моҳ (1-31)
- getDay() → гирифтани рӯзи ҳафта (0-6)
- getHours() → гирифтани соат (0-23)
- getMinutes() → гирифтани дақиқа (0-59)
- getSeconds() → гирифтани сония (0-59)
- getMilliseconds() → гирифтани миллисония (0-999)
- getTime() → гирифтани вақт аз 1 январи соли 1970 бо миллисония

### Қоида

GET = Гирифтани маълумот

---

## SET Methods

SET Methods барои иваз кардани маълумот дар Date Object истифода мешаванд.

- setDate() → иваз кардани рӯзи моҳ
- setFullYear() → иваз кардани сол
- setHours() → иваз кардани соат
- setMilliseconds() → иваз кардани миллисония
- setMinutes() → иваз кардани дақиқа
- setMonth() → иваз кардани моҳ
- setSeconds() → иваз кардани сония
- setTime() → иваз кардани вақт

### Қоида

SET = Иваз кардани маълумот

---

##  Фарқи GET ва SET

GET маълумотро мехонад ва бармегардонад.

SET маълумотро тағйир медиҳад.

Мисол:

- getFullYear() → соли ҳозираро мегирад.
- setFullYear(2030) → солро ба 2030 иваз мекунад.

---

##  Эзоҳ

Моҳҳо аз 0 сар мешаванд:

- 0 = Январ
- 1 = Феврал
- 2 = Март
- ...
- 11 = Декабр

Рӯзҳои ҳафта аз 0 сар мешаванд:

- 0 = Якшанбе
- 1 = Душанбе
- 2 = Сешанбе
- 3 = Чоршанбе
- 4 = Панҷшанбе
- 5 = Ҷумъа
- 6 = Шанбе

---

##  Хулоса

- Date барои кор бо сана ва вақт истифода мешавад.
- GET барои гирифтани маълумот истифода мешавад.
- SET барои иваз кардани маълумот истифода мешавад.
- GET ва SET аз методҳои асосии Date Object мебошанд.
